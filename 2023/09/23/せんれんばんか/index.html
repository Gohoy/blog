    千恋万花学日语笔记

    日语 千恋万花
    Created At : 2023-09-23 00:39
    Count:1.5k Views 👀 :
    Comment:
    • 千せん恋れん万ばん花か
    • お客きゃくさん　
    • なんですか　
    • 飛とぶ起おきた俺おれは、前まえに座ざつている運転うんてん手しゅに返事へんじをする
      • 飛ぶ　飞
      • 起きる　起来，立起来
      • 俺　我，俺，咱，男性对同辈或晚辈自称
      • 前（まえ）に　前方
      • 座（ざ）　座位，地位
      • 運（うん）転（てん）　开，驾驶，运转
      • 運転手　驾驶员
      • 返（へん）事（じ）　答应，回答，回话
    • そろそろ着つきますけど、どこでとめますか就快要到了，要到哪里下车呢
      • そろそろ　就要，快要，慢慢地
      • 着（つ）き：着（つ）く　抵达
      • けど　但是
      • どこ　何处
      • 止（と）める　停止
    • そうでしたね　确实（表示同意对方的话语）
    • 電車でんしゃを降おりて、駅前えきまえでタクシー捕つかまえたものの、いつの間まにかうたたねしていたらしい
      • 電車　火车
      • 降りる　下车
      • 駅前　车站旁边
      • タクシー　taxi
      • 捕まえる　抓到
      • ものの　还不到，还没有，仅仅，用不了
      • いつの間にか　不知不觉　
      • うたたね　打盹
      • ちしい　助词，表示猜测
    • まだ大雑把おおざっぱな目的もくてき地ちしか伝つたえていなかつたのを思おもいでした
      • まだ　还，仍
      • 大（おお）雑（ざつ）把（ぱ）　大概，粗略
      • 目（もく）的（てき）地（ち）　目的地
      • しか　只，仅仅
      • 伝（でん）える　转告
      • 思いで　回忆起
    • 志那しな都と荘しょうまで行くだりつて欲ほしいんですけど
      • 荘：　旅馆
      • まで：　直到…为止
      • つ：　直到
      • 欲しい：　想要，希望，愿意
      • けど：　但是
    • アタシはここいらに詳くわしくないんです
      • アタシ：　私
      • ここ：　这里
      • いら：　等等，诸如此类，等级
      • 詳しく：　详细，精通，熟悉
      • ないんです：　否定
    • ですからね…奥おくまで行いくのは勘弁かんべんしてもらえませんか 所以，能不能饶了我，不要在向里面走了
      • ですからね：　所以，因此
      • 奥：　里头，内部，尽头
      • 勘弁：　饶恕
      • 勘弁して：　饶了我吧
    • サービスの悪あくい運転うんてん手しゅだ
      • サービス：　 service
      • 悪い：　不好
    • それが顔かおに出でていたのか、運転うんてん手しゅが愛想笑あいそわらいを浮うかべながら言葉ことばを続つづける
      • 顔（かお）：　表情，脸色，脸
      • 愛（あい）想（そ）笑（わら）い：　讨好的笑，谄笑
      • 浮かべ：　出现
      • ながら：　一边，一边
      • 言（こと）葉（ば）：　语言，话语
      • 続（つづ）ける：　继续，连续
    • 本当ほんとうにすみませんえ
    • …わかりました。じゃあ、ここでいいです
    • ありがとうございます
    • タクシーが止とまり、俺おれは精算せいさんを済すませて、タクシーを降おりる
      • 止（と）まり：　停下
      • 精（せい）算（さん）：　细算，补交
      • 済（す）ませて：　完，结束
      • 降りる：　下
    • わざわざイヌシキの土地とちに祭まつりを見みに行いきるなんて、不心得ふこころえ者しゃが増ふえたもんだよ
      • わざわざ：　特地
      • イヌシキ： 游戏内专用的，犬式，表示不祥之地吧
      • 土（と）地（ち）：　地方，土地
      • 祭（まつ）り：　祭祀，祭日
      • 見（み）に：　看
      • 不（ふ）：　没有
      • 心（こころ）得（え）： 经验，知识，心得，体会
      • 者（もの）：　…的人
      • 増（ふ）える：　增加
    • 降おり間際まぎわに聞きこえてきた呟つぶやき
      • 間（ま）際（ぎわ）：　正要…的时候，快要…以前
      • 聞（き）こえる：　听得见，能听到，闻名
      • 呟（つぶや）き：　嘟哝，嘟囔
    • 無視むしして、そのまま外そとに出でた
      • 無（む）視（し）：　无视
      • した：　接着，之后
      • そのまま：　就这样
      • 外（そと）：　外面
    • まだ距離きょりがあるけど…ま、いっか。久ひさしぶりだし歩ほこう
      • 距（きょ）離（り）：　距离
      • ある：　有，持有，存在
      • いっか：　算了
      • 久しぶり：　久违，好久
    • 息いきを吐はき、両手りょうてを腰こしに当あてて、大おおきく背せをのけ反ぞらせる
      • 息（いき）：　喘气，呼吸
      • 吐（は）き：　吐出
      • 腰（こし）
      • 両（りょう）手（て）
      • 当（あ）てる：　贴上，放上，打，碰，撞
      • のけ反（ぞ）らせる：　使背向后伸展
    • ずっと座すわっていたせいか、腰こしの辺あたりでポキポキと骨ほねが鳴なった
      • せい：　的缘故
      • か：　或者
      • 辺（あた）り：　周围，大约
      • ポキポキ：　拟声词，嘎巴嘎巴
      • 骨（ほね）
      • 鳴（な）る：　（非生物）鸣，响
    • つかれた 累了
    • 家いえを出でて、電車でんしゃを乗のり継つぎ約やく２に時間じかん
      • 家（いえ）
      • 乗（の）り継（つ）ぎ：接着乘坐
      • 約（やく）
      • 時（じ）間（かｎ）
    • そしてタクシーを使つかって約やく30分ふん
      • そして：　然后
      • 使う：　使用
    • 青々あおあおとた木々きぎばかりの面白味おもしろみのない風景ふうけいの峠とうげ道どうを抜ぬけ、ようやく辿たどり着きいた穂ほ織おの町まち
      • 木々（きぎ）：　树木
      • 面（おも）白（しろ）味（み）：　愉快的
      • 風（ふう）景（けい）：　风景
      • 峠（とうげ）道（みち）：　盘山公路
      • 抜け：　穿过，溜走
      • ようやく：　渐渐的
      • 辿（たど）り着（つ）く：　挣扎地走到
      • 穂（みのる）織（おり）の町（まち）：　穗织镇
    • 相変あいかわらず不便ふべんだな、ここは
      • 相変わらず：还是老样子
      • 不便：不方便
    • 電車でんしゃも走はしっておらず、公共機関こうきょうきかんといえば路線バスろせんばすが一いち時間じかんに一いち本ほん
      • あら：在，长期存在
      • ず：助词，没有，无
      • ば：助词，…的话
      • 路線：交通线，路线
      • バス：bus
    • バスだと一いち時間じかん弱じゃくになってしまう上うえに、早朝そうちょうや日ひが暮くれる頃ころになると走はしっていない
      • たと：表示假设
      • 弱：稍微，不到
      • 上に：上句例子中第二个要点
      • 日が暮れる頃：当日落的时候
    • 他たにここを訪おとずれるためには、自分じぶんたちで車くるまを用意よういしなければならない
      • 他：别的地方
      • 訪れる：拜访
      • ためには：表示目的
      • 自分たちで：自己，亲自
      • 用意：准备
      • しなければならない：表示必须
    • 徒歩とほで移動いどうできる範囲はんいは大だい自然しぜんばかりという、山間さんかんにある田舎いなか
      • できる：能够
      • 範囲：范围
      • ばかりという：表示主要
      • 田舎：农村
    • のはずだが、その悪条件あくじょうけんのわりには中々なかなか賑にぎわいがある町まちだ
      • のはずだが：表示虽然预计不会如此
      • のわりには：表示尽管如此
      • 中々：相当
      • 賑わいがある：繁荣
    • 理由りゆうの一ひとつは、この街まち並なみ
      • のーつ：之一
      • 街並み：街景
    • 小京都しょうきょうととも呼よばれる、整備せいびされた街まち並なみは、いい意味いみで「和わ」の雰囲気ふんいきを色濃いろこく残のこしている
      • 呼ばれる：被称作
      • 整備された：整洁，整修
      • いい意味で：在积极的意义上
      • 「和」：表示和风
      • 雰囲気：气氛
      • 色濃く：浓厚的
      • 残している：被留下来
    • あの面白味おもしろみのなに山やまの風景ふうけいからは想像そうぞうできない、綺麗きれいな街まち並なみとなっている
      • となっている：成为，呈现出
    • そしてなにより、穗みのる織おの町まちは温泉おんせんが知しられている。その主おもな理由りゆうは効能こうのうだ
      • そしてなにより：而且最重要的是
      • 知られている：众所周知
    • どんな病気びょうきでも治なおるし、入浴にゅうよくすれば肌はだもシルシル、信しんじられないほ様々さまざまな効能こうのうがある万能薬ばんのうやく、とまで言いわれることがある
      • どんな病気でも：不管什么病
      • し：助词表示列举关系
      • すれば：如果
      • シルシル：表示光滑
      • 信じられないほど：难以置信的
      • 様々な：各种各样的
      • とまで：表示强烈的评价
      • 言われる：被动，被人说
      • ことがある：表示经验或存在，被认为是某种状态是有时发生的。
    • なので療養りょうよう・湯治とうじに来くる人じんが昔むかしから結構けっこういる温泉おんせん地ちだ
      • なので： 因此
      • 療養・湯治：进行疗养和泡温泉疗伤
      • 昔から：古时候
      • 結構：足够，充分，漂亮，相当
    • 近年きんねんではロコミやネットで噂うわさが広ひろまり、外国人観光客がいこくじんかんこうきゃくも増ふえたらしく、今いまも日本語にほんご以外いがいの会話かいわがたくさん聞きこえる。
      • ロコミ：地方性的传言
      • ネット：net
      • 噂：言论
      • 広まり：扩大，传播，蔓延
      • らしく：表示推测，判断
      • 今も：现今
      • たくさん：许多
    • 行いくか
    • 観光かんこうに来きたわけじゃないんだから、立たち尽つくしてても仕方しかたない
      • わけ：当然
      • だから：因此
      • 立ち尽くして：始终站着
      • 仕方ない：不是办法
    • 人違ひとちがいならごめんなさい。もしかして
      • 人違い：认错人
      • なら：如果
      • もしか：难道是
    • やっぱりよかった、人違ひとちがいじゃなくって
      • やっぱり：果然
      • じゃなくって：表示否定转折
    • やーやー、随分ずいぶんとお久ひさしぶりだね
      • 随分：相当，非常
      • 久しぶり：好久不见
    • 明あかるい笑顔えがおを浮うかべながら手てを振ふるっているこの人ひとは、馬庭まにわ蘆花ろか
    • 確たしか……俺おれより２に、３さん歳さい年上としうえだった気きがする
      • 確か：确实，记得
      • より：比，相较于
      • 気がする：心里感觉
    • の翻訳ほんやく
